## 🎤 Interview Questions

### What is a Database?

A database is an organized collection of related data.

---

### What is DBMS?

DBMS (Database Management System) is software used to create and manage databases.

Examples:
- MySQL
- Oracle
- SQL Server

---

### What is RDBMS?

RDBMS (Relational Database Management System) stores data in tables and supports relationships between tables.

Examples:
- MySQL
- PostgreSQL
- Oracle

---

### Difference between DBMS and RDBMS

| DBMS | RDBMS |
|------|--------|
| Stores data | Stores data in tables |
| Relationships not mandatory | Supports relationships |
| Less secure | More secure |

---

### What is a Primary Key?

A Primary Key uniquely identifies each record in a table.

Properties:
- Unique
- Cannot be NULL
- One primary key per table
